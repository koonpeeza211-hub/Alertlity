# Alertlity By.JustSense

OBS alert overlay สำหรับ TikTok Live (ผ่าน TikFinity)

## วิธีติดตั้ง (บรรทัดเดียว)

เปิด **PowerShell** แล้ววางคำสั่งนี้ กด Enter:

```powershell
irm https://raw.githubusercontent.com/koonpeeza211-hub/Alertlity/main/install.ps1 | iex
```

โปรแกรมจะโหลด ติดตั้งลงโฟลเดอร์ `Alertlity` บน Desktop และเปิดให้อัตโนมัติ

## วิธีติดตั้งแบบ manual

1. โหลด `Alertlity.zip` จากหน้า [Releases](../../releases/latest)
2. แตกซิปลงโฟลเดอร์ไหนก็ได้
3. เปิด `SenseLauncher.exe`

## การใช้งาน

- โปรแกรมต้องมี **license key** ถึงจะใช้งานได้ — ติดต่อซื้อได้ที่ช่องทางด้านล่าง
- คู่มือการใช้งานอยู่ในโฟลเดอร์ที่ติดตั้ง (`คู่มือการใช้งาน.md`)
